# Online Quiz System

## Project Description
The Online Quiz System is a web-based application that allows users to take quizzes online and receive instant scores.  
It provides an interactive platform for testing knowledge through multiple-choice questions.

## Features
- User-friendly quiz interface
- Multiple-choice questions
- Instant score calculation
- Displays results after quiz submission

## Technologies Used
- HTML
- CSS
- JavaScript
- (Optional: Database such as MySQL)

## How It Works
1. User opens the quiz application.
2. Questions with multiple options are displayed.
3. User selects the correct answers.
4. After submitting the quiz, the system calculates the score.
5. The final result is displayed to the user.

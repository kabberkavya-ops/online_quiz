# 🎯 Online Quiz System

## 📌 Project Overview
The Online Quiz System is a responsive web-based application designed to conduct quizzes in an interactive and efficient manner. It enables users to attempt multiple-choice questions (MCQs) and receive instant feedback based on their performance. The system is lightweight, easy to use, and ideal for academic as well as training purposes.

## 🚀 Key Features
- Clean and user-friendly interface
- Multiple-choice question (MCQ) support
- Real-time score calculation
- Instant result display after submission
- Fast and responsive design
- Easy to customize and extend

## 🛠️ Technologies Used
- HTML5 – Structure of the application
- CSS3 – Styling and layout design
- JavaScript (ES6) – Quiz logic and interactivity
- MySQL (Optional) – Data storage for questions and users

## ⚙️ Working Process
1. User opens the quiz application
2. A set of questions with multiple options is displayed
3. User selects the correct answers
4. Clicks on the Submit button
5. The system evaluates responses instantly
6. Final score and performance are displayed
